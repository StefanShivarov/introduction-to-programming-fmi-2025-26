## Оптимизирано решение, с използване на побитови операции

_(Задача от изпит по УП, 2022/23 г.)_

Да се напише функция на C++, `int analyzeStr(char *)`,
която връща като резултат броя на уникалните символи в подадения
като параметър символен низ или -1, ако низът не съществува.

Вход 1:
```
abcbabcabcbb
```

Изход 1:
```
3
```


Вход 2:
```
abc abcabc
```

Изход 2:
```
4
```
