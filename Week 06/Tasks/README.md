# Задачи от семинар, 04.11.2025

**Задача 1.** *(контролно 2, 2024/25)*

Да се напише програма, която приема от стандартния вход масив от цели числа и преобразува масива, като елементите в двете половини на масива са подредени наобратно спрямо оригиналния масив *(ако дължината на масива е нечетно число, то първата половина да е по - дългата)*.

[Виж решение](https://github.com/StefanShivarov/introduction-to-programming-fmi-2025-26/blob/main/Week%2006/Tasks/reverseHalves.cpp)

**Примери:**

Вход 1: 
```cpp
[1, 9, 4, 2, 7, 3, 6, 8] , size = 8
```
Масивът след изпълнение на ф-ята : 
```cpp
[2, 4, 9, 1, 8, 6, 3, 7]
```

Вход 2:
```cpp
[1, 9, 4, 2, 7, 3, 6] , size = 7
```
Масивът след изпълнение на ф-ята : 
```cpp
[2, 4, 9, 1, 6, 3, 7]
```
