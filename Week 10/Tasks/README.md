**Задача 1.** _(контролно 3 от минала година)_

 Напишете функция на езика на C++ с име `convertString`, която приема приема като параметри: (1) символен низ `str`, (2) символ `ch`. Функцията да преобразува низа `str`, като го обърне наобратно и всяка срещната малка буква от латинската азбука се заменя със съответната ѝ главна, а всяка главна – с малка, всяка срещната цифра се заменя с остатъка при делението ѝ на 2, а всеки символ, различен от цифра и буква, се заменя с `ch`.

Вход:

str = abcABC1234#!

ch = @

Изход:
```
@@0101cbaCBA
```

**Задача 2.** _(изпит по УП, 2022/23 г.)_

 Да се напише функция на C++, `int analyzeArr(char *)`,
която връща като резултат броя на уникалните символи в подадения
като параметър символен низ или -1, ако низът не съществува.

Вход:
```
abcbabcabcbb
```

Изход:
```
3
```
