## Задачи решени на живо - УП - практикум, 18.12.2025

**Задача 1.** _(от контролно 4 от минали години)_

Да се напише функция на С++, която приема като параметър символен низ, и връща като резултат n-тата поредна дума в низа (брои се от 1) или `nullptr`, ако такава не съществува. Дума се нарича всяка последователност от символи, която не включва интервал и/или табулация (`'\t'`). Броят на интервалите и табулациите не е ограничен.

**Задача 2.** _(от контролно 4 от минали години)_

Да се напише функция на C++ със сигнатура `int* binaryMatrixToDecimalArr(const bool* const* binary, unsigned rows, unsigned cols)`, която приема като параметър матрица от булеви елементи и за всеки ред от дадената матрица намира на колко е равно побитовото представяне на числото. Функцията да връща масив от целочислен тип с големина rows, съдържащ десетичните числа от двоичните представяния на всеки ред.

Вход:
```cpp
binary = { { 1, 0, 1 }, { 0, 1, 1 }, { 1, 1, 1 }, { 1, 0, 0 } }
rows = 4
cols = 3
```

Изход:
```cpp
[5, 3, 7, 4]
```
