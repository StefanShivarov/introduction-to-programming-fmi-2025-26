# Задачи решени на живо - УП - практикум, 08.01.2026

**Задача 1.** Да се напише функция, която приема като аргумент символен низ и връща нов символен низ, заделен с точно толкова заделена памет (дължина), колкото е необходимо. В резултатния низ за всеки различен символ, срещащ се във входния низ, трябва да се съдържа съответният символ, непосредствено последван от броя на срещанията му във входния низ, като се игнорират интервали, табулации и всякакви whitespace-ове. Всеки символ трябва да участва точно веднъж в резултатния низ. Резултатният низ трябва да бъде подреден по възходящ ASCII ред на символите.

Вход 1:
```
hello
```

Изход 1:
```
e1h1l2o1
```


Вход 2:
```
ste ste stefoto
```

Изход 2:
```
e3f1o2s3t4
```
