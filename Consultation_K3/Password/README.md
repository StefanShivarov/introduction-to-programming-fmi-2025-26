## Задача: Парола

Да се дефинира функция, която получава два параметъра - текст и низ от специални символи. Функцията трябва да провери дали текстът е валидна парола. Ако е валидна парола, да се конкатенират всички срещания на специални символи от паролата в края ѝ, в ред обратен на този, в който се срещат. Ако е невалидна нека всеки символ от паролата да стане X.

Изисквания за валидна паролата са: да е с дължина поне 8 символа и най-много 255 символа, да съдържа поне 1 главна буква, да съдържа поне 1 малка буква, да съдържа поне един от подадените специални символи.

Специалните символи не могат да бъдат букви или цифри. Ако функцията получи невалидни специални символи, трябва да приключи изпълнението си без да променя паролата.

**Пример:**

Вход (валидна парола):
```
Текст: My@Password1!--Up-SI##
Специални символи: !@-#
```
Изход:
```
My@Password1!--Up-SI####---!@
```
---
Вход (валидна парола):
```
Текст: ##My@Password1!--Up-SI
Специални символи: !#-@
```
Изход:
```
##My@Password1!--Up-SI---!@##
```
---
Вход (невалидна парола):
```
Текст: my@password1!--up-si##
Специални символи: !#-@
```
Изход:
```
XXXXXXXXXXXXXXXXXXXXXX
```
---
Вход (невалидна парола):
```
Текст: My@Weak@Pass1!
Специални символи: )&*
```
Изход:
```
XXXXXXXXXXXXXX
```
